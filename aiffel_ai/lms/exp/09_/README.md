# 의료 영상
- 사람 속을 보는 데이터
## 의료 영상의 종류
1. X-Ray
    - 밀도의 정도로 빛이 투과됨
2. CT(Computed Tomography)
    - X-Ray를 회전시켜 3D 이미지를 만듬
    - 신체의 단면 이미지를 Slice라고 부름
3. MRI(Magnetic Resonance Imaging)
    - 강한 자기장을 사용하여 신체 기관의 이미지를 찍음
# X-Ray 이미지
## 자세 분류
1. 시상면(Sagittal plane)
    - 왼쪽과 오른쪽을 나눔
2. 관상면(Cornal plane)
    - 앞뒤로 나눔
3. 횡단면(Transvese plane)
    - 상하로 나눔
- 주의점
    - 당사자 기준으로 사진을 봐야함
## X-Ray 특성
- 데이터가 명암으로 나옴
