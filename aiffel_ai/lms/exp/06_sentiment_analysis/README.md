# RNN
## 텍스트를 순서대로 인공지능에 넣는 법
### stateful vs stateless
- 과거 정보를 기억한다 vs 기억하지 않는다
- 기본적으로 RNN은 과거 기록을 사용햐여 학습한다
## RNN 말고 접근하는 방법은?
### 1-D CNN
- 단어를 묶어서 처리함
- 매우 빠름
### MaxPooling
- 가장 중요한 단어만 피처로 추출
### etc
- 1-DNN + RNN
- FFN(FeedForward Network)
- Transformer
